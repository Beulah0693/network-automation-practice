# AI Agent Guidance for network-automation-practice

- A small Python practice project focused on basic language features.
- `main.py` implements simple calculator functionality.
- `README.md` describes the repo as Python basics practice for I/O, exceptions, loops, and dictionaries.

## How to help
- Work with the existing file `main.py` and keep changes simple and aligned with a beginner Python exercise.
- Do not treat this as a full network automation application; the repo is practice code.
- If adding or improving functionality, use clear, idiomatic Python and avoid unnecessary complexity.
- Preserve or add simple examples and docstrings for readability.

## Execution
- No formal test suite exists in the repository.
- Use `python main.py` only if runner behavior is added or if manual execution is needed.

## Notes for agent behavior
- Prefer direct edits to `main.py`; there are no other source files.
- Keep the repo structure minimal and avoid introducing large frameworks or unrelated files.
- If more instructions or exercises are needed, ask the user before adding new files.
